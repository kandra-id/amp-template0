Accelerated Mobile Page atau AMP, merupakan sebuah gagasan dari raksasa mesin pencari Google agar sebuah halaman web dapat diunduh dengan cepat melalui perangkat mobile. Saat ini orang menggunakan perangkat mobile mereka untuk mencari informasi maupun untuk bertransaksi. Bagi digital marketer yang mencari pelanggan baru, membuat sebuah web valid AMP akan sangat bermanfaat. Halaman sebuah website yang dibuat menggunakan Framework AMP terbukti sangat cepat tampilnya.
